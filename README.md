# INSTALL

```
git clone git@gitlab.com:lethkhai/smart-home-be.git
cd .\smart-home-be\
```
Install all the package in the requitement.txt file
```
npm install
```
## Install dbs

```
npx sequelize-cli db:migrate
```
## Run
# Start the server
```
npm run start
```
# To publish data
```
npm run pub
```
